  • jsonplaceholder.typicode.com
  • randomuser.me
  • dummyjson.com
  • reqres.in

Tier Pro · cualquier URL + headers custom + recon extendido. Licencia ~/.api-explorer-license firmada con HMAC-SHA256.

3

Detector pesado de PII

auditoría estilo GDPR
Free

RandomUser devuelve perfiles con PII realista (mail, tel, GPS, login).

python cli.py --url "https://randomuser.me/api/" --responsable "responsable_auditoria" --cliente "Auditoría PII" --formato html
Genera salidas/auditoria-YYYYMMDD-HHMMSS.html con la lista completa de hallazgos PII clasificados por nivel (🟡 medio · 🟠 alto · 🔴 crítico).

Pipeline completa

Free

6

Diff forense entre corridas

drift detection
Free

Compará la auditoría de hoy contra una previa. Muestra campos nuevos, desaparecidos y PII que cambió.

# semana 1
python cli.py --url "https://api.cliente.com/v1" --cliente "Cliente X"

# semana 4
python cli.py --url "https://api.cliente.com/v1" --cliente "Cliente X" --diff-con salidas/auditoria-semana1.json
Diff: 1 campo nuevo (tax_id) · 1 desaparecido (legacy_codigo) · 3 PII nuevas
7

API protegida (con auth)

caso Pro · bearer o headers custom
Pro

Para cualquier URL que no esté en la lista Free, necesitás licencia Pro.

# 1) generar licencia Pro (una vez)
python scripts/generar_licencia.py --email "comprador@empresa.com" --tier pro

# 2) explorar API protegida
python cli.py --url "https://api.cliente.com/v1" --auth-bearer "TOKEN_AQUI" --cliente "Cliente X"
Headers custom (repetibles):
--auth-header "Origin:https://www.cliente.com" --auth-header "X-API-Key:abc123"
